getTimePromise

getTimePromise function to convert Swim times based on ASA table getTime(convertFrom, convertTo, stroke, timing)

based on formula documented https://www.swimmingresults.org/downloads/equivalent-time-share/algorithm.php

inputs

getTimePromise(stroke: string, convertFrom: any, convertTo: string, timing: time, covertFrom: string): promise
Parameters
stroke (string) is stroke as per the list.
convertFrom (any)
convertTo (string) is poolLength.
timing (time) is time to convert from.

returns

covertFrom (string) is poolLength.
Returns
promise: returns a promise to supply the time
Returns
time: time per target

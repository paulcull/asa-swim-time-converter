getTime

getTime function to convert Swim times based on ASA table getTime(convertFrom, convertTo, stroke, timing)

inputs

getTime(stroke: string, convertFrom: any, convertTo: string, timing: time, covertFrom: string): time
Parameters
stroke (string) is stroke as per the list.
convertFrom (any)
convertTo (string) is poolLength.
timing (time) is time to convert from.

returns

covertFrom (string) is poolLength.

time: time per target
